# Идеи по доработке форума

Краткий список того, что можно доработать, чтобы форум выглядел как сильная альтернатива текущим площадкам (Steam, Reddit, Discord и т.д.).

## Уже сделано

- **Главная тема (Featured)** — один крупный блок с изображением 21:9, градиент, заголовок и описание. Signature moment на главной.
- **Последние новости и новинки** — сетка карточек с изображениями (16:10), датой, заголовком и кратким текстом. Адаптивная сетка (1 → 2 → 4 колонки).
- **Разделы форума с превью** — у каждого раздела карточка с миниатюрой слева (на sm+), hover-масштаб изображения.

Изображения пока через placeholder (picsum.photos); при подключении API (Steam, IGDB) подставляются реальные обложки игр.

---

## Дальнейшие идеи

### Контент и изображения

1. **Реальные обложки игр** — интеграция Steam API или IGDB для автоподстановки обложек по названию игры/темы.
2. **Обложки тем** — при создании темы можно загрузить или выбрать обложку; в списке тем и на карточках показывать превью.
3. **Баннер раздела** — у категории (RPG, Экшен и т.д.) опциональный фоновый баннер на странице раздела.
4. **Галерея в посте** — в посте темы несколько изображений (галерея/слайдер), а не только одно.

### Визуал и композиция (top-design)

5. **Один «главный» блок на главной** — ротация главной темы (карусель или смена по таймеру) с разными крупными картинками.
6. **Типографика в hero** — возможность выводить название игры/темы крупным шрифтом поверх изображения (как на лендингах игр).
7. **Разные форматы карточек** — в блоке «Новости» первая карточка крупнее (featured), остальные — равная сетка.
8. **Микро-анимации** — появление карточек при скролле (stagger), плавное появление изображений (blur-up или fade).

### Адаптив и производительность (responsive-design)

9. **Responsive images** — уже используется Next/Image с `sizes`; при реальных URL добавить `srcset` под ретину (2x).
10. **Контейнерные запросы** — карточка новости меняет layout (вертикальная/горизонтальная) в зависимости от ширины контейнера, а не только viewport.
11. **Lazy-load ниже сгиба** — блок «Активные обсуждения» и вторые/третьи ряды новостей подгружать при скролле (уже даёт Next Image с lazy по умолчанию).

### Функциональность

12. **Фильтр новостей** — по жанру, по платформе, по дате.
13. **Подборка «Релизы недели»** — отдельный блок или тег для тем про свежие релизы.
14. **Рейтинг/популярное** — блок «Сейчас обсуждают» с превью и счётчиками (просмотры, ответы).

### Дополнительно

15. **Тёмная тема под «игровой» стиль** — опциональная палитра (тёмный фон, неон/акцент) для геймерской аудитории.
16. **Страница «Игра»** — отдельная сущность «игра» с обложкой, описанием и списком тем по этой игре (как в Steam).
17. **Карусель новостей на мобильном** — на узком экране блок новостей можно листать горизонтально (с видимым следующим кусочком карточки).

---

Приоритет по эффекту: 1–2 (реальные обложки), 5–6 (один сильный hero + типографика), 12–14 (фильтры и подборки). Остальное — по желанию и ресурсам.
